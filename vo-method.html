Vo Method

Hypothesis

  • External input must be converted to binary choices in order to be accurately processed by machines.
  • Not all decisions are created equal. Some decisions require strict rules while others can be made up.

Before You get started

The Vo Method uses variables, lists, if/else statements and functions. In that way a person familiar with computer code should feel at home with the Vo Method. However at its core the Vo Method is about rethinking how to break down / build up data, how to organize/ structure if/ else statements and how to decide the order in which the task is completed. In this way the Vo Method can feel like learning a new language and put your critical thinking skills to the test.

The python program provided on Github is intended for educational and research purposes. If you wish to incorporate the Vo Method into your own projects feel free to do so. However it is recommended that you build it from scratch to better fit your own requirements.

Setup

  • Download the python file from github.
  • Make sure you have python installed.
  • Open the python file and change the save_path variable to where you would like the output to be saved.
  • Create a folder named text in the same location as the save path.

User Interface

There are two text boxes name and details. Name is the name of the project and details is where you put details on how you want the task completed. There are four buttons Control HTML, Vo Method HTML and two more for you to plugin your own code and do your own research.
